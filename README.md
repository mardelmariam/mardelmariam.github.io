# The Embedded Score

This is the source repository for **The Embedded Score**, my personal technical blog about embedded systems, hardware, firmware, applied ML, and the orchestration of real-world engineering work.

The site presents selected notes, project experiences, and technical reflections related to the design and validation of real-world embedded systems.

## What I write about

The blog focuses on the practical side of building embedded systems:

- Firmware, drivers, RTOS, and bare-metal development
- PCB design, hardware bring-up, and debugging
- Sensor systems, instrumentation, and signal processing
- FPGA-assisted measurement and heterogeneous computing
- Edge AI, applied ML, and computer vision on constrained devices
- System integration, validation, and engineering trade-offs

Some posts are polished technical write-ups. Others are field notes from experiments, debugging sessions, and lessons learned the hard way.

## Why this exists

Embedded systems mastery does not have a finish line. There is always another board, another bug, another bus protocol, another timing issue, and another system that refuses to behave.

This repository keeps the site versioned, reproducible, and easy to evolve as the blog grows.
